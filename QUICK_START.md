## 📋 שלבי שימוש ראשוניים

### 1️⃣ פתח את האפליקציה בדפדפן
```
http://localhost:5173/
```

### 2️⃣ הוסף אתר WordPress ראשון
1. לחץ על "הוסף אתר חדש"
2. מלא את הפרטים:
   - שם האתר
   - כתובת URL (לדוגמה: `https://example.com`)
   - שם משתמש WordPress
   - **Application Password** (לא סיסמה רגילה!)

**איך ליצור Application Password:**
1. היכנס לאזור הניהול של WordPress שלך
2. עבור ל: **משתמשים → הפרופיל שלך**
3. גלול למטה ל-**Application Passwords**
4. הזן שם (לדוגמה: "AI Automator")
5. לחץ **Add New Application Password**
6. העתק את הסיסמה שנוצרה ✅

### 3️⃣ הגדר נושאי תוכן (Pillar Topics)
1. עבור ל"ניהול נושאים"
2. הוסף 3-5 נושאי ליבה לאתר שלך
3. לכל נושא הוסף תיאור קצר

### 4️⃣ צור תוכן ראשון עם AI
1. לחץ על "צור תוכן חדש"
2. בחר:
   - **אורך**: בינוני (מומלץ להתחלה)
   - **סגנון**: מדריך או רשימה
   - **מילות מפתח**: 2-3 מילות מפתח רלוונטיות
3. לחץ "צור רעיון לפוסט"
4. סקור את הרעיון שנוצר
5. לחץ "צור תוכן מלא"

### 5️⃣ פרסם או תזמן
- **פרסם מיד**: לחץ על כפתור הפרסום
- **תזמן לעתיד**: בחר תאריך ושעה בלוח השנה

## 🎯 תכונות מתקדמות

- מילים מועדפות ומילים להימנעות

  - שבועי (לדוגמה: כל יום ראשון)
  - חודשי (לדוגמה: ב-1 לחודש)
- סיום אוטומטי בתאריך מסוים או לאחר מספר פרסומים

### לוח שנה ויזואלי
- תצוגות: חודש, שבוע, יום, רשימה
- גרירה ושחרור לשינוי תאריכים

- צבעי קידוד לפי סטטוס

## 🔧 פתרון בעיות נפוצות

### שגיאת API Key
אם מופיעה שגיאה "API_KEY environment variable not set":
1. ודא ש-`.env.local` קיים בתיקייה הראשית
2. ודא שהמפתח מתחיל ב-`AIzaSy`
3. הפעל מחדש את שרת הפיתוח

### שגיאת חיבור ל-WordPress
- ודא שהכתובת מתחילה ב-`https://`
- ודא שאתה משתמש ב-**Application Password** ולא בסיסמה רגילה
- בדוק שהיוזר שלך הוא Administrator או Editor

### בעיות ביצועים
- נקה את ה-localStorage בדפדפן
- נקה את ה-cache של הדפדפן
- הפעל מחדש את הדפדפן
